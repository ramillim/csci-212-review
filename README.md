# CSCI 212 Review

## [Exam 1 Practice Problems](src/main/java/org/csci212/exam1)
Note that the [AlphabetizeTwoStringLists](src/main/java/org/csci212/exam1/AlphabetizeTwoStringLists.java)
problem is a more complicated than you are likely to encounter on the exam. It requires knowledge of
sorting algorithms in addition to string manipulation. It is included in case you want additional
practice.
